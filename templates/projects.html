Dynamic Resume Website

This project is a personal resume website designed to showcase my professional and technical skills, as well as my project portfolio. Below are the details of the technologies used and the architectural choices made for this project:

Technologies Used

  • Flask: A lightweight WSGI web application framework. It's chosen for its simplicity and flexibility in building web applications quickly. Flask provides the tools needed to create routes and views, which are essential for a dynamic website.
  • Python: The back-end of this site is powered by Python, leveraging its robust libraries and simplicity in handling back-end logic, including interactions with databases and rendering pages.
  • HTML/CSS/JavaScript: Used for structuring and designing the front end. HTML lays out the foundation of the pages, CSS is used for styling, and JavaScript adds interactivity to the web pages.
  • PostgreSQL: This robust, open-source relational database system is used to store and manage the site's data efficiently. It was selected for its scalability and strong community support.
  • AWS: Amazon Web Services hosts the application, providing a reliable and scalable cloud computing environment. Services used include EC2 for running the application and RDS for database hosting, ensuring high availability and performance.
  • Git/GitHub: Source code management is handled using Git, with GitHub as the remote repository. This setup facilitates version control and collaboration, allowing updates and changes to be managed systematically.

Architecture Overview

The architecture of this site is designed to be simple yet functional, suitable for a personal project but scalable to more complex deployments. The Flask application acts as the server-side engine, handling requests and responses. Data persistence is managed through PostgreSQL, ensuring that all data remains secure and consistent. The entire application is deployed on AWS, taking advantage of its robust ecosystem and scalability features.
