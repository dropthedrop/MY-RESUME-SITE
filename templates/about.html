About This Project

This project is built using a variety of modern web development tools and technologies:

  • Flask: A micro web framework written in Python.
  • Python: A high-level, interpreted programming language known for its ease of use.
  • HTML/CSS/JavaScript: Used for creating and styling the frontend of the website.
  • PostgreSQL: An open-source relational database management system.
  • AWS: Amazon Web Services for hosting and deploying the application.
  • Git/GitHub: Used for version control and managing updates to the project.
  • SQLAlchemy: ORM library for handling database operations.

This page is designed to showcase the technologies involved and how they integrate to create a fully functional dynamic website.
