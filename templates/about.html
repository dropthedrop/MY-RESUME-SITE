About This Project

This project is built using a variety of modern web development tools and technologies:

  • Flask: A micro web framework written in Python, used for building the web server and handling HTTP requests.
  • Python: A high-level, interpreted programming language known for its ease of use, serving as the backbone language for the server-side logic.
  • HTML/CSS/JavaScript: Used for creating and styling the frontend of the website, providing the structure, design, and interactive elements.
  • Docker: A platform used for containerizing the application, ensuring that it runs consistently across different computing environments.
  • AWS ECS and ECR: Amazon Web Services - Elastic Container Service for deploying the containerized application and Elastic Container Registry for storing Docker images.
  • Fargate: A serverless compute engine for containers, used with ECS to manage the underlying server infrastructure.
  • Git/GitHub: Used for version control and managing updates to the project, facilitating collaboration and tracking changes.
  • AWS CLI: Command Line Interface used to manage AWS services, enabling scripting and automation of the cloud resources.

This page is designed to showcase the technologies involved and how they integrate to create a fully functional dynamic website.
